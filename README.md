## Install

``

git clone https://github.com/MtnFranke/rpi-photo-frame
sudo sh /home/pi/rpi-photo-frame/scripts/install.sh
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker pi

``
